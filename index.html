Car Parking Status Detection using AI

 Identifying empty parking spaces in a Car Parking using camera at Lamp post view

This was my computer vision class project. Every student was required to work alone and utilize what was taught in the class to develop something unique. I developed a system that classifies parking spaces as occupied or vacant for all the spaces marked within the frame. Such a system is useful in scenarios where a camera is placed at a lamp post view and you need to keep track of how many spaces are empty for new cars to be parked. The system uses a combination of the Laplacian operator for edge detection, HAAR classifier for object recognition, and motion tracking to distinguish between parked and vacant spaces.

Code - https://github.com/ankit1khare/ComputerVision
